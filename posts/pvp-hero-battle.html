PvP Hero Battle

PvP Hero Battle is a competitive hero duel game built to be simple, fast, and skill-based. You pick a hero, fight using basic attacks and one special skill, and win by reducing your opponent's HP to 0.

Play the game

Play PvP Hero Battle →

Core gameplay

  • Real-time combat: Basic attacks and skills use cooldown timers (not turns)
  • Win condition: First hero to hit 0 HP loses
  • Skills: Each hero has one skill that deals damage and has one extra effect (stun)
  • Collection: Open packs to unlock heroes. Duplicates give shards.

Online + offline

The game supports an offline mode (vs AI) for quick battles and a server mode for room-based matches.

What’s next

I’m planning to expand the menu features like Market/Bazaar (shop + trading), improve the ranked experience, and keep balancing heroes so the game stays fair.

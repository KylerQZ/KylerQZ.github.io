Firebase setup
  1. Open Firebase Console and create a project.
  2. Enable Authentication → Email/Password.
  3. Create Firestore.
  4. Create a Web App and copy the config.
  5. Paste config into chocolet/app.js where it says FIREBASE_CONFIG_HERE.
